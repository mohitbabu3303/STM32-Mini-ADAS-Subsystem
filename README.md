# STM32-Mini-ADAS-Subsystem
Real-time ADAS prototype on STM32 (ARM Cortex-M4) featuring lane departure &amp; obstacle alerts. Utilizes FreeRTOS for deterministic task scheduling and peripheral integration (UART, SPI, I2C). Focuses on functional safety and low-latency performance. Modeled in MATLAB/Simulink for automotive-grade reliability.
